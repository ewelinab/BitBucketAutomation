# BitBucketAutomation


## Build project

```bash
python -m venv venv
source .venv/bin/activate
pip install -r requirements.txt
```


## Prerequsite

Need to add env variable either as system env, or into .env file

```bash
BITBUCKET_USERNAME=""
BITBUCKET_APP_PASSWORD=""
```
